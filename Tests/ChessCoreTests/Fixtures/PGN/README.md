# PGN Fixtures

PGN parser tests use two fixture categories:

- Synthetic PGNs authored for parser edge cases such as malformed tags,
  FEN-backed games, promotions, en passant, and export round trips.
- Small Lichess standard database samples, including games sampled from the
  January 2013 standard rated archive. Lichess database exports are released
  under Creative Commons CC0:
  https://database.lichess.org/

Do not add PGN fixtures from TWIC, commercial databases, or tournament sites
unless their redistribution license is explicit and compatible with this repo.
